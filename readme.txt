Aufgaben
=========

0.) Füge die Properties zur PresentationSettings Klasse
---------------------------------------------------------
Damit das Abspeicher der Daten über die PresentationSettings Klasse funktioniert, muss diese mittel Q_PROPERTY erweitert werden


1.) Entwickle eine PresentationTimer Klasse
-------------------------------------------
Entwickle eine Klasse namens PresentationTimer.
Folgende Eigenschaften sollte die Klasse besitzen:
- Die Dauer der Presentation
- Die Dauer der "Attention" Phase
- Die Dauer der "Finalen" Phase
- Start/Stop/Reset
- getRemainingTime (sec/min/hour)

2.) Mache die PresentationTimer Klasse in Qml sichtbar
--------------------------------------------------------
Registriere die Klasse wie PresentationSettings

3.) Initialisiere und verwende die PresentationTimer Klass in Qml
-----------------------------------------------------------------
Überall dort wo nötig: main.qml, SettingsView.qml, TimerView.qml, TimerRect.qml
